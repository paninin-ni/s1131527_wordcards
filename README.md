# 視窗（二）單字卡程式

### 專案簡介:

> 本專案提供一個具備多媒體功能的互動式英語單字學習介面。系統透過文字檔（WordCards.txt）動態載入與儲存單字資料（包含單字、音標、音檔路徑與中文解釋），並利用內建的 WindowsMediaPlayer 核心整合非同步英文音標發音播放。系統更具備鍵盤快捷監聽（Enter 鍵切換、Space 鍵重播）、計時器自動循環朗讀（Auto Play）以及彈出式強制回應編輯對話視窗功能，讓學習與內容管理更加直觀與現代化。

### 執行說明:

> 1. 資料載入與清單初始化：表單啟動時會自動讀取外部文字檔。若檔案遺失會跳出錯誤提示防呆；讀取成功則以 UTF-8 編碼解析並將單字物件動態渲染至清單控制項（lstWordList），同時在下方狀態列顯示目前的單字總數。
> 2. 單字瀏覽與互動發音：點擊清單中的單字時，主畫面文字框會即時呈現單字、音標與詳細解釋。系統會自動偵測對應路徑的 wav 或 mp3 音訊檔是否存在，若存在則立即觸發音效發音，若找不到檔案則於狀態列即時拋出提示訊息。
> 3. 自動循環朗讀（Play/Stop）：按下「自動播放」按鈕後，程式會啟動計時器（Timer）每隔固定時間自動將指針移至下一個單字進行發音。當播放超過清單顯示的一半範圍時，系統會動態修正清單的捲軸位置（TopIndex），確保正在朗讀的項目始終保持在視野中央，再次點擊即可停止。
> 4. 鍵盤快捷控制：支援免滑鼠操作，使用者在非自動播放狀態下，可直接按下鍵盤的 Enter 鍵 快速跳轉並播放下一個單字，或按下 Space 鍵 針對當前選取的單字進行重複發音聆聽。
> 5. 彈出式編輯與即時儲存：在單字清單上「雙擊滑鼠左鍵」，系統會將該索引的單字資料作為參數傳遞，並以 ShowDialog(this) 彈出式強制回應視窗（frmEditWord）開啟編輯介面。當使用者修改並點擊儲存關閉後，主視窗會重新渲染新資料，並立即將變更同步寫入更新外部文字檔。
